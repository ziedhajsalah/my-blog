## Introduction

The flex box model is composed of two type of elements: The flex container (or the parent element) and the flex items (or children elements).
An element becomes a flex container by adding a css property `display: flex` to it.

## Parent Element

The flex container properties:

- `flex-direction`: defines in which direction the container wants to stack the flex items.

  - `row` (default)

## Child elements

The flex items properties:

- `order`: specifies the order of the element. Default value is 0, means the elements are ordered like their order in the DOM.

  <div style="display: flex; width: 240px; background: green; color: white;">
    <div style="background: blue; width: 24px; margin: 4px; text-align: center; order: 1;">
      1
    </div>
    <div style="background: blue; width: 24px; margin: 4px; text-align: center;">
      2
    </div>
    <div style="background: blue; width: 24px; margin: 4px; text-align: center;">
      3
    </div>
  </div>

- `flex-grow`: specifies how much an item will grow relative to the other items. Default value is 0.

  <div style="display: flex; width: 240px; background: green; color: white;">
    <div style="background: blue; width: 24px; margin: 4px; text-align: center;">
      1
    </div>
    <div style="background: blue; width: 24px; margin: 4px; text-align: center; flex-grow: 1;">
      2
    </div>
    <div style="background: blue; width: 24px; margin: 4px; text-align: center;">
      3
    </div>
  </div>

- `flex-shrink`: specifies how much an item will shrink relative to the other items. Default value is 1. Value 0 means don’t shrink.

  <div style="display: flex; width: 240px; background: green; color: white;">
    <div style="background: blue; width: 24px; margin: 4px; text-align: center;">
      1
    </div>
    <div style="background: blue; width: 24px; margin: 4px; text-align: center; flex-shrink: 2;">
      2
    </div>
    <div style="background: blue; width: 24px; margin: 4px; text-align: center;">
      3
    </div>
    <div style="background: blue; width: 24px; margin: 4px; text-align: center;">
      4
    </div>
    <div style="background: blue; width: 24px; margin: 4px; text-align: center;">
      5
    </div>
    <div style="background: blue; width: 24px; margin: 4px; text-align: center;">
      6
    </div>
    <div style="background: blue; width: 24px; margin: 4px; text-align: center;">
      7
    </div>
    <div style="background: blue; width: 24px; margin: 4px; text-align: center;">
      8
    </div>
    <div style="background: blue; width: 24px; margin: 4px; text-align: center; flex-shrink: 0;">
      9
    </div>
    <div style="background: blue; width: 24px; margin: 4px; text-align: center; flex-shrink: 0;">
      10
    </div>
  </div>

- `flex-basis`: specifies the length (width or height following flex direction) of the item.

  <div style="display: flex; width: 240px; background: green; color: white;">
    <div style="background: blue; width: 24px; margin: 4px; text-align: center;">
      1
    </div>
    <div style="background: blue; width: 24px; margin: 4px; text-align: center; flex-basis: 120px;">
      2
    </div>
    <div style="background: blue; width: 24px; margin: 4px; text-align: center;">
      3
    </div>
  </div>

* `flex`: shorthand for `flex-grow flex-shrink flex-basis`.

* `align-self`: overrides the alignment specified in the container for `align-items` for the corresponding item
